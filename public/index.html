About the Project

Our Resume Builder is a comprehensive web application designed to help job seekers create professional, eye-catching resumes with ease. As part of our academic project, we aimed to develop a user-friendly platform that simplifies the resume creation process.

Key features include:

  • Multiple professional templates
  • Easy-to-use editing interface
  • Real-time preview
  • Export options
